# CS 联机FPS游戏项目 客户端

## 游戏介绍

本项目为FPS第一人称多人对战射击游戏，支持单人训练（尚未完善）、1v1及3v3模式。

本项目绝大部分由[Yoruko666](https://github.com/Yoruko666)独立完成，感谢[Stevezxc](https://github.com/stevezxc)在游戏早期提供了帮助。

下载Release下的最新版本到任意位置，解压并运行CS.exe即可开始游戏。

请确保客户端为最新版本，否则可能因服务器与客户端无法同步导致无法正常游玩。

联系作者：QQ 2503706199

## 版本迭代
### v1.0.0
完成了基础功能，在服务器开启的情况下，可以正常游玩1v1及3v3模式。
### v1.0.1
优化目录结构，将无用资源移出Resource，减少内存占用，提升游戏流畅度。

## 未来规划
1. 经济系统尚未完善，目前为游戏开始时一次性派发金币，击杀、胜利获取金币等功能仍待添加。
2. 目前仅有1张地图可供游玩，未来将添加更多地图随机游玩。
3. Practice单人训练模式尚未完善，直接套用多人游戏的逻辑，进入游戏后因不存在敌方队伍导致频繁开启新回合，玩家会频繁瞬移至重生点。
4. UI显示为5v5模式，实际上是3v3模式。
